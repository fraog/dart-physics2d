The physics2d library simulates simple 2d rigidbody dynamics.

Classes

Collision2D

A 2d collision object consists of an impact point and an impact normal (representing the edge that was impacted).

Edge2D

A 2d edge has a start position and a direction (which denotes the end). It is no different than a line segment.

Physics

Physics is a singleton and should not be constructed.

Rect2D

A basic 2d rectangle implementation.

RigidBody2D

A 2d rigid body.

Vector2D

A basic 2d vector implementation.
